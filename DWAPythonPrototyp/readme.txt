Diese Repository beinhaltet den Prototypen eines DWH Generators auf Basis dieser Komponenten:
- Modellierung: SqlDBM
- DWH Core Generator: Python
- DAG für Airflow: Python


Vorgehen für Ausführung des main.py:

1) Download Python Umgebung
2) Download PyCharm
3) Neues Projekt erstellen
4) Ersetze den Code des main.py durch den Code von main.py in diesem Repo 
5) Installiere alle notwendigen Python Package (z.B. pyodbc, pandas etc.)
